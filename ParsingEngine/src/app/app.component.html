Example

Possible inputs: 
        [ {"Age": 20}, { "SeniorContent" :"asd"} ]
        Or
        Any valid JSON array of objects with key value pair
Expressions: 

        A valid expression be a IF.. THEN ..ELSE type of expression

        1. if (@{Age} != "40" AND @{SeniorContent} == "asd") then "yes" else "no"
        2. if (@{Age} > "40") then "age greater" else "age smaller"
        2. if ("80" > "40") then "aaa" else "bbb"

        @{variable} will take values from the input array.
